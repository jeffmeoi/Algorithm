# 油田

## 题目
[https://vjudge.net/problem/UVA-572](https://vjudge.net/problem/UVA-572)



#### 样例输入：
```
1 1
*
3 5
*@*@*
**@**
*@*@*
1 8
@@****@*
5 5
****@
*@@*@
*@**@
@@@*@
@@**@
0 0
```
#### 样例输出:
```
0
1
2
2
```

## 题意

输入一个m行n列的字符矩阵，统计字符“@”组成多少个八连块。如果两个字符“@”所在的格子相邻（横、竖或者对角线方向），就说它们属于同一个八连块。
