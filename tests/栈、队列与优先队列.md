# 栈、队列与优先队列

## 分析

详见Cpp Standard Library.

```cpp
//栈：后进先出规则
#include<stack>
stack<int> s; 
s.push(新元素);		//入栈 
s.top();			//获取栈顶元素 
s.pop();			//出栈 
s.empty();			//判断是否为空 
s.size();			//获取栈的尺寸
//队列：先进先出原则
#include<queue>
queue<int> q;
q.push(新元素);		//入队 
q.pop();			//出队 
q.front();			//取队首元素
q.back();			//取队末元素 
q.empty();			//判断是否为空 
q.size();	 		//获取队列尺寸 
//优先队列： 可用于不断取得最值
#include<queue>
priority_queue<int> pq;//越小的整数优先级越低的优先队列
pq.push(新元素)			//入队 
pq.pop();				//出队
pq.top();				//取队首元素 
priority_queue<int, vector<int>, cmp> pq1;
//eg:
struct cmp{
	bool operator()(const int a, const int b) const {		//a的优先级比b小时返回true 
		return a % 10 > b % 10;
	}
};
priority_queue<int, vector<int>, greater<int> > pq2; 



```
